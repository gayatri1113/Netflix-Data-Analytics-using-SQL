# Netflix-Data-Analytics-using-SQL
Netflix Data Analytics using SQL – A SQL-based data analytics project that explores the Netflix dataset to uncover insights about movies, TV shows, countries, release years, ratings, directors, genres, and content trends using SQL queries.

🎬 Netflix Data Analytics using SQL
📌 Project Overview

This project focuses on analyzing the Netflix dataset using SQL to extract meaningful insights about the platform's content library. The project demonstrates SQL skills such as filtering, grouping, aggregation, sorting, and data exploration through real-world business questions.

🎯 Objectives
Analyze the distribution of Movies and TV Shows.
Identify the countries producing the most Netflix content.
Explore release year trends.
Find content by specific directors.
Determine the most common content ratings.
Analyze genres and categories.
Practice SQL queries for data analysis.
